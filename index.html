Introduction

ClassChat is painless, dead-simple communication with your classmates.

  • Basic chat doesn't require an account, making it easy to get classmates onboard
  • Conversations have a predictable, easy to find URL
  • Signing up for an account provides additional features for group chats by section, semester, and custom project groups
